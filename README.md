# Inventory-Management-App
A simple Android app to manage products, stock, and payments efficiently.


# Extended Description:

This Android application allows users to manage inventory seamlessly. You can add, update, and delete products, track stock levels, and record payments. Built in Java using Android Studio, it provides a clean interface with smooth navigation and practical functionality for small businesses or personal use.

# Technologies Used:

Java (Android Studio)

SQLite (Local database)

Android SDK

RecyclerView, Intents, and Material Design components

# Key Features:

Add, update, and delete products

Track inventory and stock availability

Record and manage payments

Search and filter products

User-friendly and responsive UI


# How to Run:

Clone this repository:

git clone <repository-link>

Open the project in Android Studio.

Build and run the app on an emulator or Android device.

The app uses SQLite for local data storage—no additional backend setup required.
